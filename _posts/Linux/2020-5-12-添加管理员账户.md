title: 添加管理员账户
---

使用root账户登录，用

```bash
useradd 用户名
```

新建用户，再用

```shell
passwd 用户名
```

设置密码。

为 `/etc/sudoers` 文件添加写权限

```shell
chmod u+w /etc/sudoers
```

打开 `/etc/sudoers` 文件，在

```shell
# User privilege specification
root    ALL=(ALL:ALL) ALL
```

下加入一句即可

```shell
# User privilege specification
root    ALL=(ALL:ALL) ALL
用户名	  ALL=(ALL:ALL) ALL
```

记得把文件权限改回来

```shell
chmod u-w /etc/sudoers
```
